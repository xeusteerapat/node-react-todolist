# node-react-todolist

Todo-list app

## Tech stacks

- Backend - [Node.js](https://nodejs.org/en/)
- Frontend - [React.js](https://reactjs.org/)
- Database - [mySQL+Sequelize ORM](https://sequelize.org/)
- Styling - [antd design](https://ant.design/)

## Start project

```bash
$ npm install
$ npm start
```

then you need to go to `client` directory and `yarn start`
